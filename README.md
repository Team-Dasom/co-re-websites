## ❤️ Introduce

> **✏️CO-RE, 코드를  다시 쓰다** : 개발자를 위한 GPT 활용 도우미
> 
코딩하다가 이것만 누가 도와줬으면 좋겠다고 생각한 적 있지 않으신가요?   
**CO-RE**는 개발과 관련된 문제들을 도와줄 수 있는 서비스를 목표로 하고 있습니다.   
ChatGPT API기반으로 **변수 추천, 코드 리팩토링, 주석 설명, 알고리즘 문제 해설**을 제공하는 기능을 가지고 있습니다.
